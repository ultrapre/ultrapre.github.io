哈洛·沙普利（Harlow Shapley）的分类法

Classification of globular clusters from physical, structural parameters has been introduced in the 1920s by Harlow Shapley and Helen Sawyer Hogg (Shapley, Sawyer (1927); Shapley 1930, II.5, pp. 11-14). They introduced Concentration Classes (sometimes referred to as Shapley-Sawyer Concentration Classes), named by numbers, typically denoted Roman, from I-XII, or 1-12, with I denoting the most concentrated or densest structure, XII the least concentrated. These concentration classes are given in these pages for each cluster. Quick overview for the 29 Messier Globular Clusters:

Class I (1): M75
Class II (2): M2, M80
Class III (1): M54
Class IV (4): M15, M28, M62, M92
Class V (7): M5, M13, M30, M53, M69, M70, M79
Class VI (1): M3
Class VII (2): M10, M22
Class VIII (3): M9, M14, M19
Class IX (3): M4, M12, M72
Class X (3): M56, M68, M107
Class X-XI (1): M71
Class XI (1): M55
Class XII (0)

我们的银河系大约有180到200个球状星团（包括29个Messier球状星团，其中两个可能只是最近才移居了）：M54和M79。 大多数其他星系也具有球状星团系统，在某些情况下（例如，对于M87而言）包含数千个球状星团！少数已知的河外小球位于较大的业余望远镜的范围内。有关示例，请参见Jim Shield的 Extragalactic Globulars网页。

虽然我们银河系中的所有球状星体以及我们的大伴星， 仙女座星系M31都已经老了，但其他 地方集团星系，如 大麦哲伦星云和 小麦哲伦星系以及 Triangulum银河M33也包含相当年轻的球形星团，可以从光谱学研究中得出肯定的结论。这些星系还包含巨大的弥散星云，其质量约为球状星团，是目前正在形成的未来年轻球状星团的明确候选者，尤其是LMC和 NGC 604的 塔兰图拉毒蛛星云NGC 2070在M33中 最近在M82（本地组以外的不规则星系）中发现了100多个年轻的球状星团。

扩散发射星云通常被称为H II区，因为它们主要由离子化氢组成，H II-表示电离能级的元素符号（此处为H）后的罗马数字：“ I”代表中性原子，“ II”在此意味着第一次电离，即氢原子失去了其单电子，对于其他元素，更高的数目（电离能级或失去的电子数）是可能的（例如He III，O III或Fe V）。

恒星形成星云NGC 3372产生了非常引人注目的奇特恒星Eta Carinae，它是我们银河系甚至宇宙中质量最大，发光最大的恒星之一。

PN

在此阶段，行星状星云（CSPN）或行星状星云核（PNN）的中心恒星是在宇宙中观测到的最热的恒星之一。观测表明，大多数中央恒星的质量范围在0.5至0.7太阳质量的狭窄范围内。

与恒星演化的时间尺度相比，行星状星云的寿命短，只有几千或一万年的时间才可见，然后随着其物质在宇宙环境中扩散而逐渐消失，使星际物质富含碳，氧气和其他元素。它的中央恒星冷却到白矮星。这就是为什么尽管在我们的银河系数千亿个星系中有很多太阳般的恒星，现在已经成熟了（特别是在球状星团中），银河系中的行星状星云的数量仅为10,000s：考虑到许多不确定性，据估计，可能只有15,000到60,000之间行星状星云（目前仅能探测到约3,000个，另一个隐藏在隐藏的星际尘埃后面）。在约150 个球状星团，每个数10万颗恒星，行星状星云被 发现只在其中4个，即皮斯1 中M15， IRAS 18333-2357在 M22，和两个最近发现的行星状星云 Jafu 1 和 Jafu 2 分别位于Palomar 6 和NGC 6441的球状星团中 （Jacoby和Fullton 1997；另请参见 George Jacoby的行星状星云画廊）。
