2019-08-13-博客搬家

我一开始用的教程，这个是基础，主要完成了主题的使用和网站的成功运行、成功链接git的本地

https://zhuanlan.zhihu.com/p/28321740

为什么用github page？markdown写作好。

用了试用主题之后push之后默认不需要上传node_modules，不用管。

打开github.io页面成功之后，使用hexo部署。我参考下面的

https://zhuanlan.zhihu.com/p/56101093

https://www.jianshu.com/p/40e7c10065c9

图片的引用我用了这篇文章，主要就是设置post_asset_folder: true和npm install https://github.com/CodeFalling/hexo-asset-image --save

https://www.jianshu.com/p/852c3eeb5377

问题是试了好多遍之后，终于成功引用了与md同名文件夹内的图片引用，同时引用也是md可见的。![](目录名/文件名.png)别写成\了。

像下面的做法，如果修改配置则全部过一遍，测试完之后最后hexo deploy是绝对没问题的，记住hexo deploy就已经上传到github了，不要再push了！！！如果只是增加了博客，那么只需要hexo g和hexo s了，有点耗时间，但是最后deploy自动不需要全部再上传了。

$ hexo clean
$ hexo generate
$ hexo server

更简单的做法 在生成网站的同时进行发布 hexo g -d

记住摘要符号的写法
