2019-09-28-git再学习

整合

大手册

https://www.yiibai.com/git/git_log.html

小索引简易

https://www.ruanyifeng.com/blog/2015/12/git-cheat-sheet.html

init新建仓库

需要注意的是暂存区提交后不会清空！

复杂点的操作比如切换commit和合并分支或者有冲突：直接右键。建议回退版本分叉新建分支。

冲突解决需要手动删除。

合并后另一个再“合并”就会覆盖。

git reset --hard HEAD回退最后一个，git reset --hard HEAD~1回退倒数第二个。

如果恢复指定文件最后加文件路径。

git log按q退出。

低level输入 高level 输入
working directory 手工创建 git checkout/git stash
stage(index) git add git reset
History(repository) git commit git pull
remote git push -

取回origin主机的master分支。$ git fetch origin master

还有remote fetch pull

git pull = git fetch + git merge

git pull –rebase = git fetch + git rebase

要添加一个新的远程仓库,可以指定一个简单的名字,以便将来引用,运行 git remote add [shortname][url]

git remote add origin XXX
