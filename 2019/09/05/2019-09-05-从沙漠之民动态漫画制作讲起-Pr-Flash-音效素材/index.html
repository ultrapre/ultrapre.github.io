2019-09-05-从沙漠之民动态漫画制作讲起-Pr-Flash-音效素材

技术向

经过两次视频制作，我的flash技术有了突飞猛进。

印象最深的是“分散到图层这个功能”，然后设置关键帧把前面的都删掉，一层一层叠起来。这是基本制作模式。

不过也有一些问题，体现在先flash后配乐会造成时间把握不准，节奏容易偏快。

Pr里面要学会设置默认的帧率（“编辑”-“首选项”-“媒体”），还有视频缩放到帧大小而不是手动调整。

音效和配乐是最麻烦的了：好好利用网上能找到的音效“战斗，战争相关的音效”和爱-给-网。

所以，先做配乐再做flash再做音效，这样工作量就会轻松一点。
