2019-03-21-端工作小报

2019-03-21-端工作小报

目的：挑选出之前打造的生产工具和自动化习惯。

#第一名：autohotkey
最大的作用就是把鼠标左边两个键变成了Ctrl+c和Ctrl+v；
alt+字母键打开应用。

#2.打造字符化端互联
上传文件到百度云用。
ahk + cmder + bgo
早已是实时git file to BDUSS的操作了，那么是否也能实现这么方便的手机端和电脑端互联呢（TIM传文件实在渣渣）。

#3.安卓端Tasker自动化
一插耳机就自动打开网易云音乐。

#4.dism系统打包
避免系统出现问题。60GB的系统盘打包成25GB文件，实时备份。Linux系统更方便直接tar。

#5.安卓Drivedroid
对于经常折腾linux不同系统的安装实在方便，直接把iso文件拷贝进手机，插上USB，开机直接选启动项爽到。大大节约了iso烧录u盘的麻烦。最最最重要的是，手机放一个PE系统iso，再也不怕系统崩盘却发现u盘没有pe系统。
