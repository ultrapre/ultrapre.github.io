2019-06-10-Autokey-for-Ubuntu（不是Autohotkey）

2019-06-10-Autokey-for-Ubuntu（不是Autohotkey）

探究linux下的热键应用，虽然没有ahk方便，但是不缺好用的；
本文就阐述一下autokey的用法。

安装：
https://github.com/autokey/autokey/releases

我一着急就把这些deb全下下来了。

dpkg -i deb安装
有时候不成功，apt –fix-broken install修复。

启动

②：命令行安装
sudo add-apt-repository ppa:cdekter/ppa
sudo apt-get update
sudo apt-get install autokey-gtk
sudo apt-get install autokey-qt

以下是使用简明教程

翻译来源：
https://www.maketecheasier.com/autokey-make-your-own-keyboard-shortcuts-in-linux/

在Windows中，我们可以使用流行的AutoHotKey来创建自定义快捷方式。那么Linux呢？ Autokey可能是最好的答案。 AutoKey是Linux和X11的桌面自动化实用程序。它允许您创建脚本并为这些脚本分配热键，允许您在任何正在使用的程序中按需执行它们。

您将在左侧窗格中看到两个名为My Phrases和Sample Scripts的文件夹。 My Phrases文件夹绑定到热键“Ctrl + F7”。 要了解Autokey的功能，打开文本编辑器并按Ctrl + F7，您应该看到带有地址选项的上下文菜单。选择“主页地址”字段时，应该会看到粘贴到文本编辑器的地址。


新建热键：

#顶级文件夹

要创建自己的热键，请转到文件 - >创建 - >新建顶级文件夹。

顶级文件夹是所有短语和脚本的容器。您可以为其分配热键并在任何应用程序中调用它。要分配热键，只需突出显示TopLevel Folder条目，然后单击Hotkey选项旁边的Set按钮。然后，您可以选择修改键（Ctrl，Shift，Alt或Super）和快捷键。

创建顶级文件夹后，接下来要创建一个短语或脚本。

#短语

短语是您经常使用的文本片段。快速按下快捷键，您可以快速将短语插入到您正在使用的文档中。 转到文件 - >创建 - >新建短语。在大文本区域中输入短语内容，并在下面的热键选项中指定快捷键。 除了分配热键外，您还可以使用短语的缩写。例如，我将缩写“mte”设置为短语“http://maketecheasier.com”。现在我只需要键入“MTE”，它将自动替换为完整的URL。

#脚本

对于那些具有脚本知识的人，您可以添加脚本，分配热键并让其在其他任何地方运行。对于那些了解脚本的人来说，“Record Marco”功能可以帮助您记录简单的键盘事件。

#结论

对于那些在键盘上花费大量时间的人来说，Autokey是一个很好的工具，可以帮助您提高工作效率。短语功能提高了您的打字速度，而脚本功能可帮助您在更短的时间内完成更多的事情。掌握脚本可能需要一些时间，但一旦被征服，我相信你不会回头看。 你试过Autokey吗？让我们知道您是如何利用它来提高生产力的。
