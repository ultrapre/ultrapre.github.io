2019-07-08-安卓深入系统分区清理总空间

自从作死刷了三星手机，My Knox再也用不上了（哭泣中）。手机沦为了单纯的电脑辅助工具和通讯工具，平时也没咋再详细玩过。

自己上网找du和df命令解释
大致-s总和、-h适合的大小格式、-m -k -g就是mb kb gb

du -s -h /data/*

15G /data/app

13G /data/data

19G /data/media

总分区

2.6G /system

48G /data

58G /mnt

19G /storage

2.6G /system

可以看到个人外存区/storage也就是平常见到的文件管理器空间才19G，显然没占多少空间，但是/data 48GB，需要大力清理。从下面几篇文章知道/data/app是apk所在，/data/data是app的缓存。/data/media貌似是/storage的挂载？（未确定）

https://www.jianshu.com/p/80e10b3dbba5

https://blog.csdn.net/itermeng/article/details/79423035

https://www.jianshu.com/p/dee33233567d

总之清理思路可以根据清理较大的/data/app也就是apk所在。
执行du -s -h /data/app/*

不方便复制可以du -s -m /data/app/* >> /sdcard/temptxt
然后adb pull /sdcard/temptxt F:/

打开，去掉M，复制到xlsx。
更改属性，排序，结果出来啦

这些就是比较大的app
使用钛备份搜索比如Xiangshe等确定卸载。

https://pan.baidu.com/s/4c6L72ko
