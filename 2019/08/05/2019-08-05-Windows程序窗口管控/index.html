2019-08-05-Windows程序窗口管控

自从上次写了windows窗口初探、尝试了调用windows api实现了窗口内嵌之后，又找到了比较新颖的玩意儿。还是关于窗口与桌面的，不过实用性好多了。

闲时总是想找点可以实现奇异功能的软件来玩玩，比如能不能把一个第三方程序嵌入到chrome的页面里面运行啥的，实现“浏览器”形态化的桌面。又想了一个点子，比如把第三方程序窗口组合成tab形式这样的？把默认的桌面概念给去掉？后来还真找到了比较丰富的玩法。

1.第一个找到的是“窗口布局”、“窗口管理”这种关键词。

https://zhuanlan.zhihu.com/p/33722847

AquaSnap，MaxTo，DisplayFusion
就上手体验来说，AquaSnap不错

移来移去的那些功能我不是很感冒，毕竟aero？也实现了。比较震惊的是下图。鼠标快捷方式。洋文不难看懂吧。这还可以把一个一般程序直接隐藏到tray托盘。上次让我倍感亲切的做法是Listary的自动记录打开文件和目录功能了，不过Listary的空白双击弹出快捷菜单不喜欢，后来才会设置，就一直保留到开机项目了。

https://www.lizhi.io/review/12741902

既然AquaSnap的布局都不让我感冒，那MaxTo就更没意义了。MaxTo适合那种需要大量工作的，稳定布局的人。

DisplayFusion花里胡哨还是有点感兴趣，不过我觉得对多屏幕更有用。比较感兴趣的是触发器和兼容性的设置。其中兼容性可以设置全局标题栏关闭这种。触发器会崩也就没多设置。

2.标签式程序管控TidyTabs

这个软件非常给力了。操作上，基本上只要记住排除chrome和学会始终显示标签即可，看下视频都会了。

https://www.lizhi.io/blog/manage-windows-with-tags-tidytabs

3.系列筛选

某些清单

这次咋主要挑没见过的看。
nTrun感觉割裂性太强，一个单独的带有UI的启动器哪里比得上万能的autohotkey呢？

QuickLook 空格键预览：个人不经常使用预览功能，跳过。

WGestures 全局鼠标手势工具：和ahk功能有冲突，况且个人是偏爱键盘操作的，鼠标就免了。

NetSpeedMonitor和TrafficMonitor:可以用，没必要。千兆网已经弱化了需要看网速来消磨时间的癖好了。

杂项——

MacType：字体美化

Fences：已经在用

Rainmeter：喧宾夺主，有空看看
