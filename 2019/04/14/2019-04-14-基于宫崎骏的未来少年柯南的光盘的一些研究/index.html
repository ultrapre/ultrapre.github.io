Conan，未来的男孩：数字图书馆由Bandai Visual（バンダイビジュアル）开发，并于1995年10月20日由Emotion Digital Software在日本发行3DO。[3] 适用于Macintosh和Windows的版本由Macromedia Director撰写，并于1995年11月25日由Emotion Digital Software在日本单独发行。[4] Pippin版本已安排在1996年，[1]但从未发布过。[5]

Director的介绍
当Bandai首次与Apple合作开发基于Mac的游戏平台，最终成为Pippin时，他们首先提出了一个简化的Macintosh Classic II，能够播放基于Director的游戏。[1] 导演，以及Apple Media Tool和MetroWerks的CodeWarrior，是苹果公司为Pippin开发推荐的平台之一。[2]
使用Macromedia Director撰写的应用程序通常（但并非总是如此）带有表示此类的徽标。Director支持使用Lingo编程语言进行回放脚本编写，允许快速原型设计和图形应用程序的开发，同时权衡创作平台的局限性。[3]
美国的Pippin @WORLD控制台与@WORLD浏览器捆绑在一起，后者支持Macromedia Director撰写的Shockwave Web内容。[4] [5]

#2.未来少年柯南游戏

设计非常精美，典型的pce游戏，被收录在pce游戏合辑中。

##运行方法
下载Ootake291.exe
发布地址：
http://www.emulator-zone.com/doc.php/pcengine/ootake.html

下载[BIOS] Super CD-ROM System (Japan) (v3.0).zip

安装Ootake291.exe，把[BIOS] Super CD-ROM System (Japan) (v3.0).pce放在C:\Program Files (x86)\Ootake，打开Ootake。
DAEMON Tools Lite运行Mirai_Shounen_Conan_(NTSC-J)_[TJCD2022].cue
点击CD-ROM game，如果出现未找到system card，则后续选择[BIOS] Super CD-ROM System (Japan) (v3.0).pce，即可运行游戏。

#光盘提取与再生模拟：
暂无提取方法。
受到提取保护，挂载后不可以使用Ultraiso，产生*cue与bin格式打开后播放“这是受到..保护的cd-rom，不可再生。。”的日语。只能说常规方法不行而已。
