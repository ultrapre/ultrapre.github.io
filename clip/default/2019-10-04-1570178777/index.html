用arc_conv推倒椎名里緒

最后更新 2013年10月4日
发表评论
教你用arc_conv解包椎名里緒系统的warc文件，取得里面的资源，图片合成，和一些收集到的解包key。
文章目录
  • 一，取得KEY
  • 二，解包
  • 三，参考资料

arc_conv是一个新出的解包工具（命令行），是俄罗斯高手w8m出品。

官方下载：http://honyaku-subs.ru/forums/viewtopic.php?f=17&t=470

解压密码请看里面的 license.txt

目前以经支持很多格式了，详细自己看 formats.txt

由于crass停止开发后，对WARC 1.7一直有很大怨念，所以这里讲一下如何推倒它。

（*）r54只支持到2.48，2.49可以用ZeaS为crass写的RioShiina插件。

（*）r55只有两个程序的升级源码，可自己运行编译。

过程其实只有两步，1，取得KEY、2，用KEY来解包

一，取得KEY

1，把arc_conv.exe改成“以管理员身份运行此程序”，把batch.rar内的warcscan.bat拿出来放到和arc_conv.exe同一目录。

2，正常运行游戏，有没破解之类不要紧，你只要运行程序就得了。
椎名里緒的EXE很霸道，要管理员才能正常运行，我用Sandboxie装和运行，免得留垃圾。

3，不要关游戏程序，现在运行warcscan.bat，得到warckey.txt，里面有一行东西，那个就是KEY。
现在关掉游戏程序，arc_conv.exe可以去掉“以管理员身份运行此程序”，如果你有很多游戏要解的话就先全部提KEY，然后再解。

注：以上是在WIN7上的过程，因为用户之间的安全机制问题要这样搞，如果在XP上的话，本来就是管理员账号，麻烦少很多。

二，解包

下面是一个实例：1.bat

Default
1
2
@echo off
arc_conv.exe -o dir --dest G:\111111\MIKOK_G1 --warc 246-AC51AC7DAC7DAC51AC7DAC7DAC51AC7DAC7DAC51 J:\MIKOK_G1.WAR

解释一下：

Default
1
arc_conv.exe -o dir

类似固定的参数，总之不要管

Default
1
--dest G:\111111\MIKOK_G1

是指明解到那个文件夹，最后一级目录不存在的话会自动建立

Default
1
--warc 246-AC51AC7DAC7DAC51AC7DAC7DAC51AC7DAC7DAC51 J:\MIKOK_G1.WAR

第一个指明解warc，第二个是KEY，第三个就是要解的WAR文件

现在运行1.bat，会有个小弹窗，里面有数字在动，YES！

RIO.INI里面写着版本，v2.37、v2.38都是不用KEY就能直接解包的。运行arc_conv.exe直接用弹窗选择WAR文件，但文件要在硬盘上，要直接解光驱上的文件要用--dest，例子：3.bat

Default
1
arc_conv.exe -o dir --dest G:\111111\DRA_G2

运行这个3.bat再用弹窗来选择光驱上的文件即可解包到G:\111111\DRA_G2目录。

图片类是 *B?.WAR、*D?.WAR、*E?.WAR、*G?.WAR

音乐类是 *M?.WAR、*SE.WAR、*V?.WAR

脚本是 *S?.WAR、*SC.WAR

对话是 *T?.WAR

注：以上是一般情况，?是可能出现的数字。

合成图片实例：2.bat （目前测试只能把两张合成为一张，参考compose_example.bat）

Default
1
arc_conv.exe --mod compose 00001.tga 00101.tga z.tga

00001.tga是在底的图
00101.tga是在面的图
z.tga输出合成的文件名

三，参考资料

转载一些资料，原文
http://www.hongfire.com/forum/showthread.php/8143-jsnut-s-Guide-to-CG-Ripping?p=2921832#post2921832

If someone want to open WARC 1.7 (RioShiina), then send me executable file from game and archive with scripts (*_S.WAR).
I have my extractor that can open these archives, but every game have its own key.
Thanks for nothing, already gathered all needed samples by myself.

Usage: Run "arc_conv.exe -o dir archive.war", or without archive name (run.bat), then you will see file selection dialog, destination folder name will be archive name plus "~".
Extractor update: http://www.mediafire.com/?w78af0wpvx7yfg4
RioShiina versions supported: v2.36 to v2.38 - any, v2.39 to v2.48 - if executable isn't protected (settec, pec).
If executable is protected and extractor can't find key, then run RioShiina application and then "arc_conv.exe --mod warcscan warckey.txt" (scan.bat), use key from "warckey.txt" with "--warc" switch.
Example: "Onedari Onapet" key = "247-9CB69CD59CA09CF69D579D779D679D7C9D790000", command line "arc_conv.exe -o dir --warc 247-9CB69CD59CA09CF69D579D779D679D7C9D790000"
