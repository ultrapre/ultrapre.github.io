转自galgame终点吧

如题，本人呢也不会编程什么的，研究各个游戏ONS移植脚本的时候
总结了几个套用公式，即使像我完全不会的，也能制作出简单的galgame游戏
因为年代久远，我也只学个皮毛，有说错的大家无视就好。

首先准备好你的素材，一个字体文件，然后建立个TXT文档准备编辑就可以了。
教程素材在这里

本帖隐藏的内容

度盘（提取码：veji）






好了下面就是开始制作的过程了，大家都知道galgame 有图片背景 语音 人物立绘等东西，下面我就开始举例


首先添加一个背景图

lsp 200,":c;bmp\h003a.jpg",0,0,255
lsp 200 是文件编号（多个图片同时显示的时候，编号小的，在前面显示，所以背景图编号要大）
c 这个我忘了，不带透明通道的用c 带透明通道的立绘什么的用a
bmp\h003a.jpg 文件目录，名字h003a图片格式为jpg在bmp文件夹内

然后在背景上添加个人物立绘




lsp 40,":a;chara\09\nan300a.bmp",140,0,255
lsp 70,":a/5,0,3;chara\09\nan300af.bmp",342,180,255:cell 70,0
因为我举例这个游戏特殊，表情和身体是分开的所以需要2条文字
lsp还是编号，a是因为立绘带透明通道，也就是黑边，chara\09是图片所在路径，图片名字为nan300a
bmp是图片格式，坐标是140,0 为不透明，
表情因为是连在一起的/5,0,3和cell 70,0，就是把这个图片竖着裁成平均5份，cell 70是和图片编号一样
0就是显示第一份（这个一般素材用不上，哪有脸和身子分开的QAQ）


有了背景和人物，现在开始添加对话
首先得有个聊天框


setwindow 40,460,27,16,26,26,0,2,40,1,1,":a;sys/win.bmp",10,430套用这个
a还是透明通道，
sys是图片所在文件夹
win.bmp是文件名字和格式
10,430是坐标
前面那些就是字体大小间距什么的，我也分不清了，没什么特殊要求直接用就好
然后添加对话
【七七】
「说纤细还不如说胆小」 \
【七七】
「人类的定义？　话说回来人不都是各不相同的吗？
」\
\为点击才能显示下一句


背景和人物，对话都有了
现在可以添加，BGM和语音了
bgm"bgm\ama13.ogg"
bgm\ama13.ogg"为bgm文件夹名字为ama13格式为ogg的文件
停止背景音乐
bgmstop
对话
dwave"voice\nan0041.ogg"
voice文件夹名字为nan0041格式为ogg的文件
停止语音
dwavestop

好现在我们有了背景 人物，语音和BGM，在填上几个命令就可以制作好了
游戏内容开头要加*title
图片显示要加print 10,1000 （否则不显示）
10是显示效果 1000是1秒
翻页效果 可以试试改成 6和7
bg black 10,1000背景黑色（背景过场什么的）
csp -1去掉所有图片
csp 200 去掉编号为200的图片，用于人物立绘的变化


;mode800 value799
*define
defsub xmove2
defsub axmove2
caption "终点制作填坑版
versionstr "1.0"
menusetwindow 25,25,2,2,0,1,#BB8866
savenumber 20
rmenu "储存进度",save,"读取进度",load,"回想模式",lookback,"快进",skip,"清除对话框",windowerase,"自动阅读",automode,"返回标题画面",reset,"退出",end
savename "保存进度","读取进度","档案"
lookbackcolor #ffff00
menusetwindow 25,25,2,2,0,1,#bb8866
menuselectcolor #ffffff,#b1b1b1,#828282
selectcolor #00ffff,#f0ffff
effectcut
nsa
windoweffect 10,200
transmode alpha
dwavefadeout 2000
effect 2,10,200rmenu_end
game
*start

*title

setwindow 40,460,27,16,26,26,0,2,40,1,1,":a;sys/win.bmp",10,430

lsp 200,":c;bmp\h003a.jpg",0,0,255
lsp 40,":a;chara\09\nan300a.bmp",140,0,255
lsp 70,":a/5,0,3;chara\09\nan300af.bmp",342,180,255:cell 70,0
print 10,1000
wait 3000
bgm"bgm\ama13.ogg"
dwave"voice\nan0041.ogg"
【七七】
「说纤细还不如说胆小」 \
dwave"voice\nan0042.ogg"
【七七】
「人类的定义？　话说回来人不都是各不相同的吗？
」\
dwavestop
bgmstop


这是整体的思路 wait 3000就是等待3秒的意思


wait可以不加，但是我怕大家一下快进内容结束退出了，所以我加了个等待3秒




剩下的我想想应该还有移动图片渐入渐出，游戏开始封面和按钮的制作，容我在坑几天，
希望对大家制作同人什么的有启发
